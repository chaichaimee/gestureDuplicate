# Повторювані жести

Виявляє та вирішує конфлікти жестів вводу у вашій конфігурації NVDA

## Опис

**Повторювані жести** — це додаток для NVDA, який дозволяє знаходити та керувати повторюваними чи конфліктними жестами вводу, а також видаляти залишки жестів, що залишилися після видалених додатків у налаштуваннях NVDA.

Він містить два потужні інструменти:

- **Перевірити повторювані жести**  
  Сканує та перелічує всі повторювані жести у всіх контекстах (глобальних модулях додатків тощо).

- **Керування моїми жестами**  
  Переглядайте та безпечно видаляйте користувацькі жести, що належать додаткам, які вже були видалені

> **Важливо**  
> Після видалення додатків багато користувацьких жестів часто залишаються у файлі `gestures.ini`, що може спричинити плутанину або конфлікти.  
> Цей додаток допоможе вам легко та безпечно їх видалити.

## Гарячі клавіші

| Комбінація                                   | Дія                                          |
|--------------------------------------------|-------------------------------------------------|
| <kbd>Windows</kbd> + <kbd>Shift</kbd> + <kbd>G</kbd> (одноразове натискання) | Відкриває діалог **Перевірити повторювані жести**       |
| <kbd>Windows</kbd> + <kbd>Shift</kbd> + <kbd>G</kbd> (подвійне натискання) | Відкриває діалог **Керувати моїми жестами**         |

Або отримати доступ через меню:  
**NVDA → Інструменти → Повторювані жести →**  
  • Перевірити повторювані жести  
  • Керувати моїми жестами

## Особливості

- Виявляє **повторювані жести** в усій NVDA (ядро + додаток)
- Чіткий, зрозумілий список, що містить: жест • назву сценарію • контекст/категорію
- Перехід одним натисканням до вбудованого діалогу NVDA **Жести вводу** із попередньо відфільтрованою назвою скрипта
- Керує **жестами, що залишилися** після раніше встановлених (тепер видалених) додатків
- Показує лише жести, які належать додаткам (ігнорує вбудовані жести NVDA)
- Жести з видалених додатків відображаються **сірим текстом**
- Видаляти окремі жести або всі жести конкретного додатка одночасно
- Функція **Очистити все** — видаляє користувацькі жести з усіх додатків за одну операцію
- Підтримує **подвійне натискання** на гарячу клавішу для швидкого перемикання між обома інструментами
- Діалоги з повною підтримкою клавіатури (підтримуються Enter, Delete, Escape)

> **Рекомендація**  
Після видалення будь-якого додатка завжди використовуйте **Керування жестами** для очищення залишкових призначень і запобігання можливим конфліктам.
